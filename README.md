This repository contains the MATLAB implementations developed as part of the EN5450 – Digital Signal Processing course project. The project focuses on the analysis and design of digital filters using various FIR and IIR filter design techniques, along with the evaluation of their performance characteristics.

The repository is organized into the following MATLAB scripts:

task1.m – Implementation and analysis of the first project task.
task2.m – MATLAB code corresponding to the second project task.
task3.m – MATLAB implementation and results for the third project task.

Each script is self-contained and includes the calculations, filter design procedures, and graphical analyses required to complete the respective task. The codes make use of MATLAB's Signal Processing Toolbox functions to design filters, compute filter parameters, and visualize the resulting impulse and frequency responses.

This repository serves as a reference for the practical application of digital signal processing concepts covered in the EN5450 course and demonstrates the use of MATLAB for solving engineering problems related to FIR and IIR filter design.
